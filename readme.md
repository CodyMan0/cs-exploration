# CS Study - Network Quest

> 네트워크의 핵심 개념을 비유와 실습으로 정복하는 프로젝트

## 진행 방식

1. 일상 비유로 개념을 함께 이해한다
2. 퀘스트(과제)를 받고 직접 풀어본다
3. 리뷰 후 **CLEAR** 또는 **RETRY** 판정
4. 언어: **C++** / **TypeScript**
